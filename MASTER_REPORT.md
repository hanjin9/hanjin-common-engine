## 1차 — CORE 메뉴 수정

### 사용자 관리 4카드

| 카드 | 클릭 진입 | 일괄 액션 |
|---|---|---|
| 전체 사용자 | 전체 명단 + 상위/하위% 필터 | 메시지/미션/이벤트/스케줄 |
| 오늘 가입 | 오늘 가입자 명단 | 환영 메시지 자동 발송 |
| 주간 활성 | 7일 내 활동자 | 응원 메시지 발송 |
| 관리자 | 관리자 명단 | 공지 발송 |

> UI: 이메일 받은편지함 방식 (전체선택 체크박스 + 일괄 액션 바)

### 멤버십 관리

| 카드 | 클릭 진입 | 액션 |
|---|---|---|
| 전체 멤버 | 전체 명단 | 일괄 발송 |
| 이달 매출 | 이번 달 결제자 명단 | 감사 메시지 |
| 활성 비율 | 최근 30일 미션 완료자 | 미션 독려 |
| 평균 AI점수 | AI점수 내림차순 명단 | 칭찬 메시지 |

- **11단계 분포**: hover → 클릭 → 티어 정책 카드 (자격조건/월회비/혜택/명단/발송)
- **프로젝트별 카드**: 클릭 → 해당 프로젝트 멤버 현황 팝업 + 타겟 발송

### 기타
- **멤버십 구매 → 멤버십 혜택**: 메뉴명 변경, 오프라인 이벤트/혜택 페이지로 전환
- **결제/정산**: Revenue/Settlement/Refund/Transaction 상세 페이지 카드 클릭 연결

## 핵심 자동화 아키텍처 (신경망 연결 설계)

```
사용자 활동
    ↓
DB 저장 (자동)
    ↓
AI 분류 엔진 (점수/등급/미션/수면/웨어러블)
    ↓
자동 트리거
    ↓
1차 피드백 발송 (격려/칭찬) ─────────→ 포인트 자동 지급
    ↓ D+3                              ↓
2차 피드백 발송 (경고/개선)        미션 자동 배정
    ↓ D+7                              ↓
3차 피드백 발송 + 코치 옵션      스케줄러 다음 사이클 예약
```

**연결 맵**:
- 미션 완료 → 포인트 자동 지급 → 격려 문자
- 웨어러블 데이터 → AI 분석 → 세그멘테이션 → 타겟 발송
- 수면 부족 3일 → 2차 피드백 트리거 → 수면 미션 배정
- 멤버십 승급 → 축하 문자 + 보너스 포인트 + 혜택 안내
- 이벤트 등록 → 캘린더 → 스케줄러 → 자동 발송
